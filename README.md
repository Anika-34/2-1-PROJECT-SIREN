# Siren – Railway E-Ticketing Platform

**Siren** is a full-stack web application designed to manage railway ticket reservations digitally. It streamlines the entire booking workflow—from user registration to ticket purchase and post-booking management—using a modern tech stack.

## Overview

The system supports passenger interactions as well as administrative operations. Users can search trains, check schedules, choose seats, make payments, request refunds (within policy limits), and leave reviews. Administrators can manage routes, schedules, train configurations, seat classes, and overall system data.

## Key Capabilities

* **Secure Authentication:** User sign-up and login functionality.
* **Train Search & Booking:** Route-based search with real-time seat availability.
* **Dynamic Seat Selection:** Interactive seat selection with automatic fare calculation.
* **Payment Handling:** Integrated (demo) payment workflow.
* **User Dashboard:** Access booking history, refunds, and reviews.
* **Admin Control Panel:** Add trains, define routes, configure stops, classes, and seat counts.
* **Data Initialization:** Pre-structured datasets (Excel/CSV) with Python scripts to generate import-ready files for the database.

## Technology Stack

* **Frontend:** React
* **Backend:** Node.js
* **Database:** PostgreSQL
* **Data Processing:** Python
